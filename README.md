# 📰 Fake News App

A cross-platform Flutter application for detecting and identifying fake news. Built with Dart and Flutter, the app runs on Android, iOS, Web, macOS, Linux, and Windows.

---

## 🚀 Features

- 🔍 Fake news detection — analyze news articles and headlines for credibility
- 📱 Cross-platform support — runs on mobile, web, and desktop
- ⚡ Fast and responsive UI built with Flutter
- 🧪 Includes a test suite for quality assurance

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| [Flutter](https://flutter.dev/) | UI framework |
| [Dart](https://dart.dev/) | Programming language |
| CMake / C++ | Native desktop platform layers |
| Swift | iOS native integration |

---

## 📁 Project Structure

```
fake-news-app/
├── android/        # Android platform configuration
├── ios/            # iOS platform configuration
├── lib/            # Main Dart/Flutter source code
├── linux/          # Linux desktop platform layer
├── macos/          # macOS desktop platform layer
├── web/            # Web platform configuration
├── windows/        # Windows desktop platform layer
├── test/           # Unit and widget tests
├── pubspec.yaml    # Flutter dependencies and project metadata
└── analysis_options.yaml  # Dart static analysis rules
```

## ⚙️ Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (stable channel recommended)
- Dart SDK (bundled with Flutter)
- A connected device or emulator / browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shahadkhraisheh96/fake-news-app.git
   cd fake-news-app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   # Mobile (Android/iOS)
   flutter run

   # Web
   flutter run -d chrome

   # Desktop (macOS example)
   flutter run -d macos
   ```
